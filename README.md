# horas-de-producao
App para calcular as horas de produção para empresa de engarrafamento de água mineral. 

Cálculo leva em conta os seguintes dados para efetuar o cálculo de horas de produção:
- Consumo do Dia
- Volume Atual
- Volume de Segurança
- Vazão do Poço
